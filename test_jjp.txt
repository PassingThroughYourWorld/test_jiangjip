	（10）、解：执行顺序 C 申明变量$num为-1，进入if语句，该判断语句可解析为 ：
		$num ++ && （$num += 2 && （$num += 3）） => 首先判断$num ++ 先显示再自加为-1，判断为真，再进入下一步，此时$num=0；  再算$num += 3 => $num = 0+3 = 3； 再$num += 2 && 3，先判断，此时2&&3为真显示为1，结果为$num += 1，$num = 3+1 = 4；-1 && 4 为真，进入if里面，再自加1，此时$num为5；
		所以$num的结果为5；
